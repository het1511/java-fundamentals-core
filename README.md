# Java Fundamentals

A hands-on Java learning repository covering core Java concepts from the ground up — built as part of a structured 2-month intensive study plan targeting a junior Java developer role.

## What This Repo Covers

| Week | Topic |
|------|-------|
| Week 1 | OOP — Classes, Constructors, Encapsulation, Exception Handling |
| Week 2 | Collections, Streams, Java 8+ Features |
| Week 3 | Multithreading, File I/O, JUnit 5 |
| Week 4 | Design Patterns, SOLID Principles, Maven, Git |

## Project Structure

```
src/
├── Main.java          # Entry point
├── Person.java        # OOP fundamentals — Week 1
└── ...                # More classes added weekly
```

## Concepts Demonstrated

### Week 1 — OOP & Exception Handling
- `Person.java` — private fields, public constructor, getters, setters with validation, `toString()`
- Custom exception throwing with `IllegalArgumentException`
- try-catch exception handling

## How to Run

1. Clone the repo
   ```bash
   git clone https://github.com/<your-username>/java-fundamentals.git
   ```
2. Open in IntelliJ IDEA
3. Make sure JDK 17+ is configured
4. Run `Main.java`
